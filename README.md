# SuperMart Billing System using Python

The SuperMart Billing System is a Python-based console application developed to simulate a real-world supermarket billing process. The primary objective of this project is to automate bill generation, calculate product totals, apply discounts, and generate a detailed customer receipt efficiently. This project demonstrates the practical implementation of core Python programming concepts in a business-oriented scenario.

The application begins by greeting the customer and displaying a list of products available in the supermarket, including their prices and measurement units. Customers can select products by entering the corresponding item number and specify the quantity required. The program then calculates the individual product cost based on predefined rates stored in a dictionary data structure.

To enhance functionality and maintain organized records, the system stores item details, quantities, and calculated prices using Python lists. The application continuously accepts multiple product selections until the customer chooses to stop the purchasing process. Once all items are entered, the program automatically calculates the total bill amount.

An additional feature included in the project is a discount system, where customers receive a 10% discount if their total purchase exceeds Rs 1000. This demonstrates the use of conditional statements and logical operations in solving real-world billing requirements.

The project also generates a detailed receipt that includes the customer’s name, purchase date and time, item-wise billing information, total amount, discount applied, and final payable amount. The datetime module is used to display the current transaction date and time, making the receipt more realistic and professional.

Technologies and concepts used in this project include Python programming, loops, conditional statements, dictionaries, lists, user input handling, arithmetic operations, and the datetime module. This project helped strengthen problem-solving skills and provided practical exposure to developing business automation applications using Python.

The SuperMart Billing System serves as a strong beginner-level portfolio project that showcases programming fundamentals, logical thinking, and the ability to develop real-world applications.
